# Parse-React-Questions-Answers

Simple Q&A app with React.js and Parse

#Built with
* react
* flux
* react-router
* parse
* parse-react

## setup
```
npm install
```

## run the project
```
npm start
```

## insert keys from your parse.com account
```javascript
// Insert your app's keys here:
Parse.initialize('APPLICATION_ID', 'JAVASCRIPT_KEY');
```
